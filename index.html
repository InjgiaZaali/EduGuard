Project pitch

EduGuardian solves exam stress by turning large study materials into short explanations, quizzes, flashcards, and a personal revision plan. It is useful for students because it shows what to study, how to test yourself, and which weak areas need more practice.

Prototype built for an innovation project. Later version can connect to a real AI API.
